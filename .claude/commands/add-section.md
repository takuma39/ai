AI駆動開発.md に新しいセクションを追加します。

追加トピック: $ARGUMENTS

## 実行手順

### ステップ1: 調査
**research-agent** サブエージェントを呼び出し、「$ARGUMENTS」について最新情報を調査してください。

### ステップ2: 現状把握
AI駆動開発.md の現在の構成（目次・見出し一覧）を確認し、以下を決定：
- 新セクションの最適な挿入位置
- 関連する既存セクションとの論理的な流れ
- 適切な見出し階層（`##` / `###` / `####`）

### ステップ3: セクション作成
CLAUDE.md で規定された以下のテンプレートに従ってセクションを作成：

1. **概要説明**（1〜2文）
2. **比較表またはフロー図**（Mermaid推奨）
3. **具体的なプロンプト例またはコード例**（`text` または適切な言語タグで囲む）
4. **ポイント・注意事項**（`>` ブロック）

### ステップ4: 整合性確認
- 「各フェーズの成果物と完了条件」テーブルの更新が必要か確認し、必要であれば更新
- 前後のセクションとの文体・トーンの一致を確認
- 既存セクションとの内容重複がないか確認

### ステップ5: レビュー
作成後、**doc-reviewer** サブエージェントを呼び出して追加セクションの品質チェックを実行してください。

## 注意事項

- CLAUDE.md の Markdown スタイルガイドに厳密に従うこと
- 見出し階層は挿入位置に応じて適切に設定すること
- 必ず日本語で作成すること
- 数値・仕様で出典が不明な場合は「〜目安」「〜程度」と明記すること
- コードブロックには必ず言語タグを付けること
- 専門用語は初出時に説明を補足すること
