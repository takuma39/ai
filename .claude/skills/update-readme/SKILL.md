README.md を現在のスキル構成・コマンド定義に同期します。

スキルの追加・削除・変更を反映し、README のコマンド表・エージェント表・Mermaid図を最新状態に更新します。

## 実行手順

**readme-updater** サブエージェントを呼び出し、以下を実行させる：

1. `README.md`・`CLAUDE.md`・`.claude/skills/*/SKILL.md`・`.claude/agents/*.md` を全て読む
2. 差分を検出（追加・削除・変更）
3. README の該当セクションのみ更新（他のセクションは変更しない）
4. 変更サマリーを報告

## 出力形式

### README 更新サマリー

#### 変更内容
（追加/削除/変更されたコマンド・エージェントのリスト）

#### 更新箇所
（変更されたセクション名と概要）

#### 差分なしの場合
「README は最新状態です」と報告して終了

## 注意事項

- README の文体・構成は維持し、データ部分のみ更新する
- 差分がない場合は何も編集しない
- 大幅な構成変更が必要な場合はユーザーに確認を求める
